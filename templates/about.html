About Us

Welcome to our Loan Eligibility Prediction System. This project is designed to assist financial institutions and individuals in making informed decisions regarding loan approvals using Machine Learning.

Project Objective

The goal of this system is to automate the evaluation of loan applications based on important financial and personal indicators. The system can predict loan eligibility for a single applicant or a batch of applications using a file upload.

Technologies Used

  • Python & Flask for backend logic and API
  • XGBoost for Machine Learning model
  • HTML, CSS for frontend design
  • JupyterLab for model development
  • VS Code for full-stack integration

Why This Project?

Loan approval processes often require time and manual effort. This system saves time, reduces human error, and ensures consistency in the decision-making process. It can also be a great learning resource for students and developers interested in real-world ML applications.
